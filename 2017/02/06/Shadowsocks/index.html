使用方法

这里为您提供了自动化地配置文件生成，包含了所有 Shadowsocks 服务器的信息，方便您在诸多的服务器中快速添加，快速切换。

Windows 下载 C# 版

  • 下载这个， 将文件解压到桌面或硬盘内，xp系统请使用2.0版本，win7至win10使用4.0版本。

  • 服务器后面的数字在 服务器Ip 内，远程端口t后面的数字填在 服务器端口 内，加密方式在 加密 中，按指定的选择即可。其余不用理会。

  • 以上信息需与下方节点信息一一对应，否者很可能无法上网。

  • 然后选择一个合适的服务器，更新一下PAC为绕过国内IP，然后开启统代理为PAC模式即可翻墙上网。

Mac OS X

  • 下载这个 安装，

  • 然后下载这个 运行程序，小飞机上右键 服务器列表 子菜单 的 “导入服务器配置文件…” 导入这个文件，然后选择一个合适的服务器，更新一下PAC，然后开启系统代理即可上网。

laptop_mac iOS

  • ios官方苹果商店 强烈推荐下载Shadowrocket，就可以批量添加节点。

Android下载

  • Android 版，点击这个 安装，下方有两种安装方式。

方法一：

  • 配置名称随便写，服务器填写节点信息中的服务器后面的数字，远程端口填写下方节点信息中远程端口后面的数字，密码填写下方节点信息对应的密码后面的数字，加密方法在下方节点信息中加密方式后面，直接选择即可。其余不需理会。
  • 以上信息切记需与下方信息一一对应，否者很可能连接后无法上网。
  • 添加完节点，然后路由选择绕过大陆，点击右上角粉色小飞机开启就可以上网了。

方法二（推荐）：

  • 点击粉框的shadowsocks R标题，点击右下角粉色加号，选择扫描二维码，扫描下方节点信息中任意一种二维码即可。
  • 扫描二维码后，选择扫码之后新增的一串数字的那个，然后右上角点击粉色小飞机开启就可以翻越城墙，探索新世界了。
